Ответы на вопросы по HTML

1. Как создать простейшую веб-страницу?

Простейшая веб-страница создается с помощью базовой структуры HTML-документа:

<!DOCTYPE html> <html> <head> <title>Заголовок страницы</title> </head> <body> <h1>Привет, мир!</h1> <p>Это моя первая веб-страница.</p> </body> </html>

2. Что представляет собой структура HTML-документа?

Структура HTML-документа состоит из:

  • <!DOCTYPE html> - объявление типа документа
  • <html> - корневой элемент
  • <head> - раздел с метаинформацией
  • <body> - основное содержимое страницы

3. Дайте определение понятиям «тег» и «элемент» HTML-документа?

Тег - это слова или символы, с помощью которых браузер распознает структуру и значение вашего текста (например, <p>, <div>).

Элемент - это тег вместе с его содержимым и атрибутами (например, <p class="text">Текст</p>).

4. Какие теги относятся к служебным, а какие – к структурным?

Служебные теги: <!DOCTYPE>, <html>, <head>, <title>, <meta>, <link>, <style>, <script>.

Структурные теги: <body>, <header>, <nav>, <main>, <section>, <article>, <aside>, <footer>, <div>.

5. В каком разделе страницы указывается заголовок веб-страницы?

Заголовок веб-страницы указывается в разделе <head> с помощью тега <title>.

6. Для чего используются метаданные? Каким тегом они указываются в структуре HTML-документа?

Метаданные используются для предоставления информации о странице (кодировка, описание, ключевые слова, автор и т.д.). Они указываются с помощью тега <meta> в разделе <head>.

7. Для чего используется !DOCTYPE?

<!DOCTYPE> указывает браузеру, какая версия HTML используется в документе, чтобы браузер мог правильно отображать страницу.

8. Каким образом можно изменить установки для всей страницы в целом?

Установки для всей страницы можно изменить с помощью:

  • CSS-стилей в теге <style> в разделе <head>
  • Глобальных атрибутов в теге <body>
  • Изменения метатегов в разделе <head>

9. Перечислите виды списков, которые предусмотрены в HTML?

В HTML предусмотрены следующие виды списков:

  • Маркированный список - <ul>
  • Нумерованный список - <ol>
  • Список определений - <dl>

10. Как изменить вид маркера в маркированном списке?

Вид маркера можно изменить с помощью CSS-свойства list-style-type:

ul { list-style-type: square; /* circle, disc, none */ }

11. Поясните, как изменить вид маркера в нумерованном списке.

Вид маркера в нумерованном списке можно изменить с помощью CSS-свойства list-style-type:

ol { list-style-type: upper-roman; /* decimal, lower-alpha, upper-alpha etc. */ }

12. Каким образом изменить нумерацию в нумерованном списке?

Нумерацию можно изменить с помощью атрибута start или CSS-счетчиков:

<ol start="5"> <li>Пункт 5</li> <li>Пункт 6</li> </ol>

13. Из каких частей состоит список с определениями?

Список определений состоит из:

  • <dl> - контейнер для списка определений
  • <dt> - термин
  • <dd> - определение термина

14. Каким образом можно построить вложенный список?

Вложенный список создается путем размещения одного списка внутри элемента <li> другого списка:

<ul> <li>Пункт 1 <ul> <li>Подпункт 1.1</li> <li>Подпункт 1.2</li> </ul> </li> <li>Пункт 2</li> </ul>

15. Создайте вложенный список из нумерованного, состоящего из двух пунктов, в каждом из которых два маркированных подпункта. Символ маркера квадрат.

<ol> <li>Пункт 1 <ul style="list-style-type: square;"> <li>Подпункт 1.1</li> <li>Подпункт 1.2</li> </ul> </li> <li>Пункт 2 <ul style="list-style-type: square;"> <li>Подпункт 2.1</li> <li>Подпункт 2.2</li> </ul> </li> </ol>

16. С помощью каких тегов создается простая таблица?

Простая таблица создается с помощью тегов:

  • <table> - контейнер таблицы
  • <tr> - строка таблицы
  • <td> - ячейка таблицы
  • <th> - заголовочная ячейка

17. Для чего предназначен тег <th>?

Тег <th> предназначен для создания заголовочных ячеек таблицы. Текст в таких ячейках по умолчанию выделяется жирным шрифтом и выравнивается по центру.

18. Какие элементы определяют строку таблицы?

Строку таблицы определяет тег <tr> (table row).

19. Что такое гипертекст?

Гипертекст - это текст, содержащий гиперссылки на другие документы или части того же документа. HTML (HyperText Markup Language) является языком разметки гипертекста.

20. Как организовать гиперссылку?

Гиперссылка организуется с помощью тега <a> с атрибутом href, указывающим URL-адрес:

<a href="https://example.com">Текст ссылки</a>

21. Каким образом организовать переход в начало или конец документа?

Для перехода внутри документа используются якоря с идентификаторами:

<a href="#top">В начало</a> ... <h1 id="top">Заголовок</h1>

22. Создайте таблицу с гиперссылкой на другую веб-страницу.

<table> <tr> <th>Название</th> <th>Ссылка</th> </tr> <tr> <td>Пример сайта</td> <td><a href="https://example.com">example.com</a></td> </tr> </table>

23. Каким образом организовать гиперссылку, используя графическое изображение?

Для создания графической гиперссылки нужно поместить тег <img> внутрь тега <a>:

<a href="https://example.com"> <img src="image.jpg" alt="Описание изображения"> </a>

24. Какие теги называются семантическими? Назовите основные семантические теги.

Семантические теги - это теги, которые четко описывают свое содержание и роль на странице. Основные семантические теги:

  • <header> - верхний колонтитул
  • <nav> - навигация
  • <main> - основное содержание
  • <section> - раздел документа
  • <article> - независимая часть content
  • <aside> - боковая панель
  • <footer> - нижний колонтитул

25. Для чего необходимы теги <section> и <article>?

<section> используется для группировки тематического контента, обычно с заголовком.

<article> предназначен для независимого, самодостаточного контента, который может распространяться отдельно (статьи, записи блога и т.д.).

26. Какую функцию выполняют теги <footer> и <header>?

<header> содержит вводную информацию или навигационные ссылки (обычно в верхней части страницы или раздела).

<footer> содержит информацию о авторе, copyright, контакты и т.д. (обычно в нижней части страницы или раздела).

27. Для чего предназначены теги <nav> и <main>?

<nav> предназначен для основных навигационных ссылок по сайту.

<main> содержит основной уникальный контент страницы (не должен повторяться на других страницах сайта).

28. Какие элементы необходимы для построения формы?

Для построения формы необходимы:

  • <form> - контейнер для формы
  • <input> - поле ввода
  • <textarea> - многострочное поле ввода
  • <select> и <option> - выпадающий список
  • <button> или <input type="submit"> - кнопка отправки
  • <label> - метка для поля ввода

29. Назовите атрибут тега <input>, который определяет вид элемента формы.

Атрибут type определяет вид элемента формы.

30. Какие значения принимает атрибут type?

Атрибут type может принимать следующие значения:

  • text - текстовое поле
  • password - поле для пароля
  • email - поле для email
  • number - поле для чисел
  • checkbox - флажок
  • radio - переключатель
  • submit - кнопка отправки
  • reset - кнопка сброса
  • button - обычная кнопка
  • file - поле для загрузки файла
  • date - поле для выбора даты
  • и другие
