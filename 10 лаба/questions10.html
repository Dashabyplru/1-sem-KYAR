Адаптивный дизайн, Flexbox и медиазапросы

Ответы на вопросы по адаптивной верстке, Flexbox и медиазапросам

Типы верстки веб-страниц

1. Раскройте суть фиксированной верстки веб-страницы?

Фиксированная верстка (fixed layout) — это подход, при котором ширина страницы задается в пикселях и не изменяется в зависимости от размера окна браузера или устройства.

Характеристики фиксированной верстки:

  • Фиксированная ширина контейнера (обычно 960px, 1024px или 1200px)
  • Элементы имеют абсолютные размеры в пикселях
  • Макет выглядит одинаково на всех устройствах
  • На маленьких экранах появляется горизонтальная прокрутка
  • Проще в разработке и тестировании

Пример фиксированной верстки (ширина 960px):

Фиксированный контейнер 960px
.container {
  width: 960px;
  margin: 0 auto;
}

/* Все элементы внутри имеют фиксированные размеры */
.sidebar {
  width: 200px;
  float: left;
}

.content {
  width: 740px;
  float: right;
}

2. Что такое резиновый макет веб-страницы?

Резиновый макет (fluid layout) — это подход, при котором ширина элементов задается в процентах относительно родительского контейнера, что позволяет макету "растягиваться" и "сжиматься" в зависимости от размера окна браузера.

Характеристики резиновой верстки:

  • Ширина элементов задается в процентах (%)
  • Макет адаптируется к ширине окна браузера
  • Используются относительные единицы измерения
  • На очень широких экранах может выглядеть растянуто
  • На очень узких экранах может стать нечитаемым

Пример резиновой верстки (ширина 80%):

Резиновый контейнер 80%
.container {
  width: 80%; /* Контейнер занимает 80% ширины окна */
  margin: 0 auto;
}

.sidebar {
  width: 25%; /* Сайдбар занимает 25% ширины контейнера */
  float: left;
}

.content {
  width: 75%; /* Контент занимает 75% ширины контейнера */
  float: right;
}

3. Дайте определение понятия «адаптивный дизайн»

Адаптивный дизайн (responsive design) — это подход к веб-дизайну, при котором веб-страница автоматически подстраивается под различные устройства и размеры экранов, обеспечивая оптимальное отображение и удобство использования.

Ключевые принципы адаптивного дизайна:

  1. Гибкая сетка - использование относительных единиц (%, fr, em, rem)
  2. Гибкие изображения - масштабирование изображений относительно контейнера
  3. Медиазапросы - применение разных стилей в зависимости от характеристик устройства

Пример адаптивного контейнера:

Адаптивный контейнер 100%
/* Базовые стили для мобильных устройств */
.container {
  width: 100%;
  padding: 15px;
}

/* Для планшетов */
@media (min-width: 768px) {
  .container {
    width: 90%;
    margin: 0 auto;
  }
}

/* Для десктопов */
@media (min-width: 1200px) {
  .container {
    width: 1140px;
    margin: 0 auto;
  }
}

Медиазапросы

4. Что такое медиазапросы? Как подключить медиазапросы?

Медиазапросы (media queries) — это правила CSS, которые позволяют применять стили в зависимости от характеристик устройства или окна браузера (ширина, высота, ориентация, разрешение и т.д.).

Способы подключения медиазапросов:

  1. Внутри CSS-файла с помощью @media: 
    @media (max-width: 768px) {
  .element {
    color: red;
  }
}
  2. В HTML через атрибут media в теге link: 
    <link rel="stylesheet" media="(max-width: 768px)" href="mobile.css">
  3. В HTML через тег style: 
    <style>
  @media (max-width: 768px) {
    body { font-size: 14px; }
  }
</style>

Демонстрация медиазапросов:

Этот блок меняет цвет фона в зависимости от ширины окна:

  • По умолчанию: серый фон
  • При ширине более 560px: зеленый фон
  • При ширине менее 960px: красный фон

Попробуйте изменить размер окна браузера!

5. Для чего используется правило @import?

Правило @import используется для импорта внешних CSS-файлов в текущую таблицу стилей. Это позволяет разделять стили на несколько файлов и подключать их по мере необходимости.

Важно: @import может замедлять загрузку страницы, так как файлы загружаются последовательно. Для повышения производительности рекомендуется использовать несколько тегов <link> или сборщики.

/* Импорт базовых стилей */
@import url("reset.css");

/* Импорт стилей для печати */
@import url("print.css") print;

/* Импорт с медиазапросом */
@import url("mobile.css") (max-width: 768px);

/* Импорт шрифтов */
@import url('https://fonts.googleapis.com/css2?family=Roboto&display=swap');

6. С какой целью применяется правило @media?

Правило @media применяется для создания медиазапросов — условных блоков CSS, которые применяются только при выполнении определенных условий (характеристик устройства).

Основные цели использования:

  • Создание адаптивного дизайна
  • Оптимизация отображения для разных устройств
  • Создание отдельных стилей для печати
  • Учет ориентации устройства (альбомная/портретная)
  • Учет разрешения экрана (ретина-дисплеи)

Примеры использования @media:

/* Для мобильных устройств */
@media (max-width: 767px) {
  .menu { display: none; }
  .mobile-menu { display: block; }
}

/* Для планшетов */
@media (min-width: 768px) and (max-width: 1023px) {
  .container { width: 90%; }
}

/* Для печати */
@media print {
  .sidebar, .ads { display: none; }
  body { font-size: 12pt; }
}

/* Для ретина-дисплеев */
@media (-webkit-min-device-pixel-ratio: 2), (min-resolution: 192dpi) {
  .logo { background-image: url('logo@2x.png'); }
}

8. Что означает @media (min-width: 560px)?

Медиазапрос @media (min-width: 560px) означает, что CSS-правила внутри этого блока будут применяться только когда ширина окна браузера (viewport) составляет 560 пикселей или больше.

Наглядное объяснение:

@media (min-width: 560px) {
  /* Эти стили применятся когда:
     - Ширина окна = 560px ✓
     - Ширина окна = 800px ✓
     - Ширина окна = 1200px ✓
     - Ширина окна = 400px ✗ (слишком мало)
  */
  .container {
    width: 80%;
    margin: 0 auto;
  }
}

Этот медиазапрос часто используется для создания "mobile-first" дизайна, где базовые стили для мобильных, а дополнительные — для более широких экранов.

9. Расшифруйте запись @media (max-width: 960px).

Медиазапрос @media (max-width: 960px) означает, что CSS-правила внутри этого блока будут применяться только когда ширина окна браузера (viewport) составляет 960 пикселей или меньше.

Наглядное объяснение:

@media (max-width: 960px) {
  /* Эти стили применятся когда:
     - Ширина окна = 960px ✓
     - Ширина окна = 768px ✓
     - Ширина окна = 480px ✓
     - Ширина окна = 1200px ✗ (слишком много)
  */
  .sidebar {
    display: none; /* Скрываем сайдбар на маленьких экранах */
  }
  
  .content {
    width: 100%; /* Растягиваем контент на всю ширину */
  }
}

Этот медиазапрос часто используется для адаптации десктопного дизайна под планшеты и мобильные устройства.

Flexbox

7. Для чего используется свойство flex?

Свойство flex является сокращенным свойством для flex-grow, flex-shrink и flex-basis. Оно определяет, как flex-элемент будет изменять свои размеры для заполнения доступного пространства.

Синтаксис: flex: flex-grow flex-shrink flex-basis;

Примеры использования свойства flex:

flex: 1;
flex: 2;
flex: 1;
.item1 { flex: 1; }   /* grow=1, shrink=1, basis=0% */
.item2 { flex: 2; }   /* grow=2, shrink=1, basis=0% - в 2 раза шире */
.item3 { flex: 1; }   /* grow=1, shrink=1, basis=0% */

/* Эквивалентная запись без сокращения */
.item1 {
  flex-grow: 1;
  flex-shrink: 1;
  flex-basis: 0%;
}

10. Что означает flex: 1 2 200px;?

Запись flex: 1 2 200px; расшифровывается как:

  • flex-grow: 1 - элемент может увеличиваться пропорционально другим элементам
  • flex-shrink: 2 - элемент будет сжиматься в 2 раза быстрее других элементов при нехватке места
  • flex-basis: 200px - начальный размер элемента составляет 200 пикселей

Визуализация:

flex: 1 2 200px;
flex: 1 1 200px;
.element {
  flex: 1 2 200px;
  /* Эквивалентно: */
  flex-grow: 1;
  flex-shrink: 2;
  flex-basis: 200px;
}

При уменьшении контейнера первый элемент будет сжиматься быстрее (в 2 раза), так как у него flex-shrink: 2.

11. Для чего предназначено значение wrap?

Значение wrap для свойства flex-wrap позволяет flex-элементам переноситься на следующую строку, если они не помещаются в одну строку внутри flex-контейнера.

Демонстрация flex-wrap:

Элемент 1
Элемент 2
Элемент 3
Элемент 4
Элемент 5
Элемент 6
.flex-container {
  display: flex;
  flex-wrap: wrap; /* Элементы будут переноситься на новую строку */
}

/* Другие значения flex-wrap */
.flex-container {
  flex-wrap: nowrap; /* Без переноса (по умолчанию) */
  flex-wrap: wrap; /* С переносом */
  flex-wrap: wrap-reverse; /* С переносом в обратном порядке */
}

Попробуйте уменьшить ширину окна браузера, чтобы увидеть перенос элементов!

12. С какой целью используется свойство flex-flow?

Свойство flex-flow является сокращением для flex-direction и flex-wrap. Оно позволяет задать направление и перенос элементов в одной записи.

Примеры использования flex-flow:

/* Направление - строка, с переносом */
.flex-container {
  flex-flow: row wrap;
}

/* Направление - колонка, без переноса */
.flex-container {
  flex-flow: column nowrap;
}

/* Направление - строка в обратном порядке, с переносом */
.flex-container {
  flex-flow: row-reverse wrap;
}

/* Эквивалентная запись без сокращения */
.flex-container {
  flex-direction: row;
  flex-wrap: wrap;
}

13. Для чего применяется свойство flex-direction?

Свойство flex-direction определяет направление, в котором flex-элементы располагаются внутри flex-контейнера.

Демонстрация flex-direction:

Элемент 1
Элемент 2
Элемент 3
/* Основные значения flex-direction */
.flex-container {
  flex-direction: row; /* Слева направо (по умолчанию) */
  flex-direction: row-reverse; /* Справа налево */
  flex-direction: column; /* Сверху вниз */
  flex-direction: column-reverse; /* Снизу вверх */
}

В демонстрации выше используется flex-direction: column, поэтому элементы расположены вертикально.

14. С какой целью используются свойства align-items и свойство align-content?

align-items и align-content — это свойства для выравнивания flex-элементов, но они работают в разных ситуациях:

align-items - выравнивает элементы внутри каждой строки по поперечной оси (cross axis). Работает всегда.

align-content - выравнивает сами строки внутри контейнера по поперечной оси. Работает только когда есть несколько строк (при flex-wrap: wrap).

Демонстрация align-items:

Высокий
Низкий
Высокий
/* Выравнивание элементов внутри строки */
.flex-container {
  align-items: stretch; /* Растянуть (по умолчанию) */
  align-items: flex-start; /* К началу */
  align-items: flex-end; /* К концу */
  align-items: center; /* По центру */
  align-items: baseline; /* По базовой линии текста */
}

15. Какие значения имеет свойство align-self?

Свойство align-self позволяет переопределить выравнивание отдельного flex-элемента, установленное свойством align-items для всего контейнера.

Допустимые значения:

  • auto - наследует значение от align-items контейнера (по умолчанию)
  • flex-start - выравнивание к началу поперечной оси
  • flex-end - выравнивание к концу поперечной оси
  • center - выравнивание по центру поперечной оси
  • baseline - выравнивание по базовой линии текста
  • stretch - растягивание элемента

Демонстрация align-self:

align-self: flex-start
align-self: center
align-self: flex-end
.flex-container {
  align-items: flex-start; /* Все элементы выравниваются к началу */
}

.item1 {
  align-self: flex-start; /* Остается у начала */
}

.item2 {
  align-self: center; /* Переопределяется на центр */
}

.item3 {
  align-self: flex-end; /* Переопределяется на конец */
}

16. Назовите свойства, которые имеет flex-контейнер.

Flex-контейнер (элемент с display: flex) имеет следующие свойства:

  1. flex-direction - направление элементов
  2. flex-wrap - перенос элементов
  3. flex-flow - сокращение для direction и wrap
  4. justify-content - выравнивание по главной оси
  5. align-items - выравнивание по поперечной оси
  6. align-content - выравнивание строк (при наличии нескольких строк)

17. Какими свойствами обладают flex-элементы?

Flex-элементы (дочерние элементы flex-контейнера) имеют следующие свойства:

  1. order - порядок отображения
  2. flex-grow - коэффициент увеличения
  3. flex-shrink - коэффициент сжатия
  4. flex-basis - начальный размер
  5. flex - сокращение для grow, shrink и basis
  6. align-self - индивидуальное выравнивание

18. Для чего предназначено свойство justify-content?

Свойство justify-content определяет выравнивание flex-элементов по главной оси (main axis) внутри контейнера.

Демонстрация justify-content:

Элемент 1
Элемент 2
Элемент 3
/* Выравнивание по главной оси */
.flex-container {
  justify-content: flex-start; /* К началу (по умолчанию) */
  justify-content: flex-end; /* К концу */
  justify-content: center; /* По центру */
  justify-content: space-between; /* Равномерно, первый и последний у краев */
  justify-content: space-around; /* Равномерно, с отступами по краям */
  justify-content: space-evenly; /* Равномерно, с равными отступами */
}

В демонстрации выше используется justify-content: space-between.

20. Чем отличается свойство align-items от свойства align-self?

align-items - устанавливается для flex-контейнера и применяется ко всем flex-элементам внутри него. Определяет выравнивание элементов по поперечной оси.

align-self - устанавливается для отдельного flex-элемента и переопределяет значение align-items, установленное для контейнера. Позволяет задать индивидуальное выравнивание для конкретного элемента.

Наглядное сравнение:

/* Контейнер с общим выравниванием */
.flex-container {
  display: flex;
  align-items: center; /* Все элементы по центру */
}

/* Переопределение для конкретного элемента */
.special-item {
  align-self: flex-start; /* Этот элемент будет у начала, не смотря на align-items: center */
}

21. Как создать адаптивное меню?

Адаптивное меню меняет свой вид в зависимости от размера экрана. Чаще всего на десктопе это горизонтальное меню, а на мобильных устройствах — "гамбургер-меню".

Пример адаптивного меню:

  • Главная
  • О нас
  • Услуги
  • Портфолио
  • Контакты
<style>
  /* Базовые стили - для мобильных */
  .responsive-menu ul {
    flex-direction: column;
    display: none; /* Скрываем на мобильных */
  }
  
  /* Кнопка гамбургер-меню для мобильных */
  .menu-toggle {
    display: block;
  }
  
  /* Для планшетов и десктопов */
  @media (min-width: 768px) {
    .responsive-menu ul {
      display: flex;
      flex-direction: row;
      justify-content: space-around;
    }
    
    .menu-toggle {
      display: none; /* Скрываем кнопку на больших экранах */
    }
  }
</style>

<nav class="responsive-menu">
  <button class="menu-toggle">☰ Меню</button>
  <ul>
    <li><a href="#">Главная</a></li>
    <li><a href="#">О нас</a></li>
    <li><a href="#">Услуги</a></li>
    <li><a href="#">Портфолио</a></li>
    <li><a href="#">Контакты</a></li>
  </ul>
</nav>

Попробуйте изменить размер окна браузера! На мобильных (меньше 768px) меню станет вертикальным.

22. Создайте документ, в котором цвет абзаца изменяется на красный при изменении ширины до 768 px.

Решение задачи:

Этот абзац меняет цвет при ширине окна меньше 768px

Текущая ширина окна: 0px
