Основы CSS

1. Что такое CSS? Как расшифровывается CSS?

CSS (Cascading Style Sheets) - это каскадные таблицы стилей. Это язык стилей, который определяет внешний вид HTML-документов, включая макет, цвета, шрифты и другие визуальные аспекты.

2. Что представляет из себя синтаксис CSS?

Синтаксис CSS состоит из селектора и блока объявлений:

селектор {
  свойство: значение;
  свойство: значение;
}

Например:

p {
  color: blue;
  font-size: 16px;
}
3. Как подключаются внутренние таблицы стилей?

Внутренние таблицы стилей подключаются с помощью тега <style> в разделе <head> HTML-документа:

<head>
  <style>
    body {
      background-color: #f0f0f0;
    }
  </style>
</head>
4. Каким образом создаются внешние таблицы стилей?

Внешние таблицы стилей создаются в отдельном файле с расширением .css и подключаются с помощью тега <link>:

<head>
  <link rel="stylesheet" href="styles.css">
</head>

Содержимое файла styles.css:

body {
  font-family: Arial, sans-serif;
  margin: 0;
  padding: 20px;
}
5. Что такое строковое подключение стилей?

Строковое подключение стилей - это применение CSS-правил непосредственно к HTML-элементу с помощью атрибута style:

<p style="color: red; font-size: 18px;">Этот текст красного цвета и размером 18px.</p>

Селекторы и специфичность

6. Как создаются классы?

Классы создаются в CSS с помощью точки (.) и применяются к HTML-элементам с помощью атрибута class:

/* В CSS */
.highlight {
  background-color: yellow;
  padding: 5px;
}

/* В HTML */
<p class="highlight">Этот текст будет выделен.</p>
7. Каким образом создаются идентификаторы?

Идентификаторы создаются в CSS с помощью решетки (#) и применяются к HTML-элементам с помощью атрибута id:

/* В CSS */
#header {
  background-color: #333;
  color: white;
  padding: 20px;
}

/* В HTML */
<div id="header">Заголовок сайта</div>
8. Как подключается универсальный стиль?

Универсальный стиль применяется ко всем элементам на странице с помощью селектора *:

* {
  margin: 0;
  padding: 0;
  box-sizing: border-box;
}
14. Что означает групповой селектор и как он создается?

Групповой селектор позволяет применять одинаковые стили к нескольким элементам одновременно. Создается путем перечисления селекторов через запятую:

h1, h2, h3 {
  font-family: 'Arial', sans-serif;
  color: #333;
  margin-bottom: 15px;
}
17. Что такое специфичность селектора?

Специфичность - это вес селектора, который определяет, какое CSS-правило будет применено к элементу при наличии конфликтующих правил. Специфичность рассчитывается на основе комбинации селекторов:

  • Идентификаторы: 100 очков за каждый
  • Классы, псевдоклассы и атрибуты: 10 очков за каждый
  • Элементы и псевдоэлементы: 1 очко за каждый

Например, селектор #content .article p имеет специфичность 100 + 10 + 1 = 111.

Взаимодействие элементов

9. Что такое дочерний элемент?

Дочерний элемент - это прямой потомок родительского элемента (первый уровень вложенности). В CSS выбирается с помощью селектора >:

/* Выберет только прямые дочерние элементы div в section */
section > div {
  border: 1px solid #ccc;
}
10. Раскройте суть понятия «родительский элемент».

Родительский элемент - это элемент, который непосредственно содержит другие элементы (дочерние элементы). Например, в структуре <div><p>Текст</p></div> элемент div является родительским для элемента p.

11. Что такое родственные элементы?

Родственные элементы (сиблинги) - это элементы, имеющие общего родителя и находящиеся на одном уровне вложенности. В CSS выбираются с помощью селекторов + и ~:

/* Выберет первый параграф, следующий сразу за h2 */
h2 + p {
  font-weight: bold;
}

/* Выберет все параграфы, следующие после h2 */
h2 ~ p {
  color: #666;
}

Каскадность и наследование

12. В чем заключается каскадность стилей?

Каскадность - это принцип, по которому браузер определяет, какие CSS-правила применять к элементу, когда есть конфликтующие правила. Браузер учитывает:

  1. Важность правила (!important)
  2. Специфичность селектора
  3. Порядок следования правил (последнее правило имеет приоритет)
13. Каков смысл наследования стилей?

Наследование позволяет дочерним элементам получать некоторые стили от своих родительских элементов. Это уменьшает количество повторяющегося кода. Например, если установить цвет текста для body, все текстовые элементы внутри унаследуют этот цвет, если для них не указано другое значение.

15. Какие свойства не наследуемые?

Некоторые CSS-свойства не наследуются дочерними элементами. Вот некоторые из них:

margin
padding
border
background
width
height
position
display
float
overflow

Цвета в CSS

16. Перечислите способы, которыми можно задать цвет в CSS.

В CSS цвет можно задать несколькими способами:

По имени: red
HEX: #FF0000
RGB: rgb(255, 0, 0)
RGBA: rgba(255, 0, 0, 0.7)
HSL: hsl(120, 100%, 50%)
HSLA: hsla(120, 100%, 50%, 0.7)

Демонстрация цветных букв CSS

18. Создайте на веб-странице текст «CSS» и первую букву выделите красным цветом; вторую букву – зеленым, третью – синим цветом.

На этой странице уже реализовано это требование. В заголовке вы видите слово "CSS", где:

  • Первая буква "C" выделена красным цветом
  • Вторая буква "S" выделена зеленым цветом
  • Третья буква "S" выделена синим цветом

Это реализовано с помощью следующих стилей:

.first-letter {
  color: #ff4d4d;
}

.second-letter {
  color: #4dff4d;
}

.third-letter {
  color: #4d4dff;
}
