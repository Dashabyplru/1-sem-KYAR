Основы CSS3: Псевдоклассы, псевдоэлементы и работа с текстом

1. Что такое псевдоэлементы? Назовите псевдоэлементы, которые вы знаете.

Псевдоэлементы — это селекторы, которые позволяют задать стиль определенной части элемента, не относящейся к элементам документа.

Известные псевдоэлементы:

  • ::first-letter — применяет стиль к первой букве текста
  • ::first-line — применяет стиль к первой строке текста
  • ::before — создает псевдоэлемент и добавляет его перед содержимым выбранного элемента
  • ::after — создает псевдоэлемент и добавляет его после содержимого выбранного элемента
  • ::selection — позволяет применить стили к элементам, которые были выделены пользователем
  • ::marker — позволяет применить стили к маркерам списка
  • ::placeholder — позволяет применить стили к тексту атрибута placeholder в элементах input и textarea

2. Раскройте суть понятия «псевдоклассы».

Псевдоклассы — это селекторы, которые определяют динамическое состояние элементов, которое изменяется с помощью действий пользователя, а также в зависимости от положения в DOM.

Примеры псевдоклассов:

  • :hover — применяется при наведении курсора на элемент
  • :active — применяется в момент активации элемента
  • :focus — применяется к элементу, получившему фокус
  • :first-child — выбирает первый дочерний элемент
  • :nth-child() — выбирает элементы по их порядковому номеру
  • :checked — применяется к отмеченным элементам формы

3. Для чего используется псевдоэлемент ::before? Приведите пример.

Псевдоэлемент ::before используется для создания псевдоэлемента и добавления его перед содержимым выбранного элемента.

Пример CSS:
.element::before {
  content: "→ ";
  color: #3498db;
}

4. С какой целью применяется псевдоэлемент ::after? Приведите пример.

Псевдоэлемент ::after используется для создания псевдоэлемента и добавления его после содержимого выбранного элемента.

Пример CSS:
.element::after {
  content: " *";
  color: #e74c3c;
}
Текст элемента

5. Каким образом изменить первую букву текста?

Для изменения первой буквы текста используется псевдоэлемент ::first-letter.

Пример CSS:
p::first-letter {
  font-size: 2em;
  color: #e74c3c;
  font-weight: bold;
}

Это пример текста с измененной первой буквой. Обратите внимание, как первая буква выделяется размером и цветом.

6. Как создать отступ первой строки?

Для создания отступа первой строки используется свойство text-indent.

Пример CSS:
p {
  text-indent: 2em;
}

Это пример текста с отступом первой строки. Обратите внимание, что первая строка начинается с отступа, а последующие строки выравниваются по левому краю.

7. Расскажите, как убрать отступы между фрагментами текста?

Для удаления отступов между фрагментами текста используется свойство margin с нулевым значением.

Пример CSS:
p {
  margin: 0;
}

8. Каким образом изменить регистр?

Для изменения регистра текста используется свойство text-transform.

Пример CSS:
.uppercase {
  text-transform: uppercase;
}
.lowercase {
  text-transform: lowercase;
}
.capitalize {
  text-transform: capitalize;
}

9. Поясните, как сделать малые прописные буквы.

Для создания малых прописных букв используется свойство font-variant со значением small-caps.

Пример CSS:
.small-caps {
  font-variant: small-caps;
}

10. Как добавить тень к тексту?

Для добавления тени к тексту используется свойство text-shadow.

Пример CSS:
.text-shadow {
  text-shadow: 2px 2px 4px rgba(0,0,0,0.3);
}

Для изменения цвета маркера нумерованного списка можно использовать псевдоэлемент ::marker.

Пример CSS:
ol li::marker {

13. Как добавить новый шрифт?

Для добавления нового шрифта используется правило @font-face.

Пример CSS:
@font-face {
  font-family: 'MyFont';
  src: url('myfont.woff2') format('woff2'),
       url('myfont.woff') format('woff');
}
body {
  font-family: 'MyFont', sans-serif;
}

14. В каких форматах добавляется новый шрифт и от чего это зависит?

Шрифты добавляются в различных форматах, чтобы обеспечить совместимость с разными браузерами:

  • WOFF/WOFF2 — современные форматы, обеспечивающие хорошее сжатие и широкую поддержку браузерами
  • TTF/OTF — стандартные форматы шрифтов
  • EOT — формат для старых версий Internet Explorer
  • SVG — формат для мобильных устройств

Выбор формата зависит от требований к совместимости с браузерами и размера файла.

15. Как изменить маркер на изображение для маркированного списка?

Для изменения маркера на изображение используется свойство list-style-image.

Пример CSS:
ul {
  list-style-image: url('marker.png');
}

Альтернативный способ с использованием псевдоэлемента ::before:

Пример CSS:
ul.custom {
  list-style: none;
}
ul.custom li::before {
  content: "";
  display: inline-block;
  width: 16px;
  height: 16px;
  background-image: url('marker.png');
  background-size: contain;
  margin-right: 8px;
}

16. В каких единицах можно задавать параметры текста?

Параметры текста можно задавать в следующих единицах измерения:

  • Абсолютные единицы: px (пиксели), pt (пункты), cm (сантиметры), mm (миллиметры), in (дюймы)
  • Относительные единицы:
    • em — относительно размера шрифта родительского элемента
    • rem — относительно размера шрифта корневого элемента (html)
    • % — процент от размера родительского элемента
    • vw/vh — процент от ширины/высоты окна просмотра
    • ch — ширина символа "0" текущего шрифта
