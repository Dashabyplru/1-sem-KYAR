📦 Блочные и строчные элементы в CSS

1. Дайте понятие «блочный элемент». Приведите примеры.

Блочный элемент — это элемент, который занимает всю доступную ширину родительского контейнера и начинается с новой строки.

Характеристики блочных элементов:

  • Занимают 100% ширины родителя
  • Начинаются с новой строки
  • Могут содержать другие блочные и строчные элементы
  • Можно задавать ширину, высоту, отступы и границы
Примеры блочных элементов:
<div>, <p>, <h1>-<h6>, <section>, <article>, <header>, <footer>, <ul>, <ol>, <li>

2. Раскройте суть понятия «строчный элемент». Приведите примеры.

Строчный элемент — это элемент, который занимает только необходимую ширину для своего содержимого и не начинает новую строку.

Характеристики строчных элементов:

  • Занимают только необходимую ширину
  • Не начинают новую строку
  • Нельзя задавать ширину и высоту
  • Вертикальные отступы (margin-top, margin-bottom) не работают
Примеры строчных элементов:
<span>, <a>, <strong>, <em>, <img>, <input>, <label>

3. Как задать свойства блочных элементов строчным?

Чтобы строчному элементу задать свойства блочного, используется свойство display со значениями:

  • display: block; — превращает в блочный элемент
  • display: inline-block; — создает строчно-блочный элемент
Пример преобразования строчного элемента:
span {
  display: inline-block;
  width: 150px;
  height: 50px;
  margin: 10px;
}

4. Что включает в себя сокращенная запись свойства margin?

Сокращенная запись margin может принимать от 1 до 4 значений:

  • margin: 10px; — все 4 стороны: 10px
  • margin: 10px 20px; — верх/низ: 10px, право/лево: 20px
  • margin: 10px 20px 15px; — верх: 10px, право/лево: 20px, низ: 15px
  • margin: 10px 20px 15px 5px; — верх: 10px, право: 20px, низ: 15px, лево: 5px

5. Какие значения имеет свойство box-shadow?

Свойство box-shadow принимает следующие значения:

  • offset-x — горизонтальное смещение тени
  • offset-y — вертикальное смещение тени
  • blur-radius — радиус размытия тени
  • spread-radius — радиус распространения тени
  • color — цвет тени
  • inset — внутренняя тень (необязательно)
Примеры box-shadow:
box-shadow: 5px 5px 10px rgba(0,0,0,0.3);
box-shadow: 0 0 20px #3498db;
box-shadow: inset 0 0 10px #000;

6. Как разместить элемент по центру?

Существует несколько способов центрирования элементов:

Центрирование блочного элемента:
.center-block {
  width: 200px;
  margin: 0 auto;
}
Центрирование с помощью Flexbox:
.container {
  display: flex;
  justify-content: center;
  align-items: center;
}

7. Какие свойства имеют блочные элементы?

Блочные элементы поддерживают все CSS-свойства, включая:

  • Размеры: width, height, min-width, max-width, min-height, max-height
  • Отступы: margin, padding
  • Границы: border, border-radius
  • Фон: background, background-color, background-image
  • Позиционирование: position, top, right, bottom, left, z-index
  • Тень: box-shadow
  • Переполнение: overflow

8. Перечислите свойства, которые имеет граница блочных элементов.

Свойства границы (border):

  • border-width — толщина границы
  • border-style — стиль границы
  • border-color — цвет границы
  • border-radius — скругление углов
  • Отдельные стороны: border-top, border-right, border-bottom, border-left
Сокращенная запись border:
border: 2px solid #3498db;

9. Каким образом определяется ширина блочного элемента?

Ширина блочного элемента определяется по формуле:

Общая ширина = width + padding-left + padding-right + border-left + border-right + margin-left + margin-right

При box-sizing: border-box формула меняется:

width = содержание + padding + border

Пример расчета ширины:
.element {
  width: 300px;
  padding: 20px;
  border: 5px solid #000;
  margin: 10px;
}
/* Общая ширина: 300 + 20*2 + 5*2 + 10*2 = 370px */

10. Для чего используется свойство box-sizing? Какие значения принимает это свойство?

Свойство box-sizing определяет, как рассчитываются ширина и высота элемента.

Значения:

  • content-box (по умолчанию) — width/height включают только содержимое
  • border-box — width/height включают содержимое, padding и border
Пример использования:
* {
  box-sizing: border-box;
}

Это значение упрощает расчеты размеров элементов.

11. Для чего используется свойство overflow?

Свойство overflow управляет отображением содержимого, которое выходит за границы элемента.

Значения:

  • visible — содержимое отображается за пределами элемента
  • hidden — обрезает содержимое за границами элемента
  • scroll — добавляет полосы прокрутки
  • auto — добавляет полосы прокрутки только при необходимости
overflow: visible
Этот текст выходит за границы элемента и отображается полностью.
overflow: hidden
Этот текст обрезается за границами элемента и не отображается.
overflow: scroll
Этот текст имеет полосы прокрутки всегда, даже если они не нужны.
overflow: auto
Этот текст имеет полосы прокрутки только при необходимости.

12. Что обозначают значения в следующем объявлении: border-color: yellow red green blue;?

В сокращенной записи border-color с 4 значениями цвета распределяются по часовой стрелке:

  • yellow — верхняя граница
  • red — правая граница
  • green — нижняя граница
  • blue — левая граница

13. Какое свойство можно использовать, чтобы задать ширину только правой границы?

Для задания ширины только правой границы используется свойство border-right-width.

Пример:
border-right-width: 5px;

Или в сокращенной форме:

border-right: 5px solid #3498db;
