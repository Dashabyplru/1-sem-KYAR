Селекторы атрибутов, псевдоклассы и работа с таблицами

1. Что представляет из себя селектор атрибутов?

Селектор атрибутов позволяет применять стили к элементам с конкретными атрибутами и их значениями.

Это мощный инструмент для выбора элементов на основе их атрибутов, что особенно полезно для форм, ссылок и изображений.

2. Какие типы селекторов атрибутов вы знаете?

Основные типы селекторов атрибутов:

  • [attr] — выбирает элементы с указанным атрибутом
  • [attr="value"] — выбирает элементы с точным значением атрибута
  • [attr^="value"] — выбирает элементы, у которых значение атрибута начинается с указанной строки
  • [attr$="value"] — выбирает элементы, у которых значение атрибута заканчивается указанной строкой
  • [attr*="value"] — выбирает элементы, у которых значение атрибута содержит указанную подстроку
  • [attr~="value"] — выбирает элементы, у которых значение атрибута содержит указанное слово
  • [attr|="value"] — выбирает элементы, у которых значение атрибута равно указанной строке или начинается с нее

3. Расскажите, как убрать отступы между ячейками таблицы.

Для удаления отступов между ячейками таблицы используется свойство border-collapse со значением collapse.

Пример CSS:
table {
  border-collapse: collapse;
}

4. Как убрать пустые ячейки таблицы?

Пустые ячейки можно скрыть с помощью свойства empty-cells со значением hide.

Пример CSS:
table {
  empty-cells: hide;
}

Это свойство работает только когда border-collapse: separate.

5. Что означают значения в сокращенной записи свойства padding?

Сокращенная запись свойства padding может принимать от 1 до 4 значений:

  • padding: 10px; — все стороны
  • padding: 10px 20px; — верх/низ и право/лево
  • padding: 10px 20px 15px; — верх, право/лево, низ
  • padding: 10px 20px 15px 5px; — верх, право, низ, лево (по часовой стрелке)

6. Для чего используется псевдокласс :focus?

Псевдокласс :focus применяется к элементам, которые получили фокус (например, при клике мышью или переходе по Tab).

7. С какой целью применяется псевдокласс :nth-child(odd)?

Псевдокласс :nth-child(odd) выбирает все нечетные дочерние элементы.

Пример CSS:
tr:nth-child(odd) {
  background-color: #f2f2f2;
}

8. Для чего предназначен псевдокласс :nth-child(even)?

Псевдокласс :nth-child(even) выбирает все четные дочерние элементы.

Пример CSS:
tr:nth-child(even) {
  background-color: #f9f9f9;
}

9. Каким образом сделать тень к элементам формы?

Для добавления тени к элементам формы используется свойство box-shadow.

Пример CSS:
input, button {
  box-shadow: 2px 2px 5px rgba(0,0,0,0.2);
}

10. Как изменить промежуток между ячейками?

Для изменения промежутка между ячейками используется свойство border-spacing.

Пример CSS:
table {
  border-collapse: separate;
  border-spacing: 10px;
}

11. Для чего используется свойство border-radius?

Свойство border-radius используется для скругления углов элементов.

Пример CSS:
.rounded {
  border-radius: 15px;
}
Элемент со скругленными углами

12. Какое свойство используется для выравнивания текста в таблице по вертикали в ячейках?

Для вертикального выравнивания текста в ячейках таблицы используется свойство vertical-align.

13. Назовите свойство, которое предназначено для выравнивания текста в таблице по горизонтали в ячейках.

Для горизонтального выравнивания текста в ячейках таблицы используется свойство text-align.

14. Какие значения может принимать свойство vertical-align?

Свойство vertical-align может принимать значения:

  • baseline — выравнивание по базовой линии
  • top — выравнивание по верхнему краю
  • middle — выравнивание по середине
  • bottom — выравнивание по нижнему краю
  • sub — нижний индекс
  • super — верхний индекс
  • Процентные значения и единицы длины

15. Для чего используется псевдокласс :first-child?

Псевдокласс :first-child выбирает первый дочерний элемент родителя.

Пример CSS:
tr:first-child {
  font-weight: bold;
}

16. С какой целью применяется псевдокласс :last-child?

Псевдокласс :last-child выбирает последний дочерний элемент родителя.

Пример CSS:
tr:last-child {
  border-bottom: 2px solid #000;
}

17. Для чего предназначены теги <thead>, <tfoot> и <tbody>?

Эти теги семантически разделяют таблицу на логические части:

  • <thead> — заголовок таблицы (шапка)
  • <tfoot> — подвал таблицы
  • <tbody> — основное тело таблицы

Это улучшает доступность и позволяет отдельно стилизовать разные части таблицы.

18. Какому свойству относится значение separate и для чего оно используется?

Значение separate относится к свойству border-collapse.

Оно используется для отображения ячеек таблицы с раздельными границами и позволяет управлять расстоянием между ячейками через border-spacing.

19. Какому свойству относится значение collapse и для чего оно используется?

Значение collapse также относится к свойству border-collapse.

Оно объединяет границы соседних ячеек в одну общую границу, убирая промежутки между ячейками.

20. Для чего предназначены теги <colgroup> и <col>?

Теги <colgroup> и <col> используются для стилизации столбцов таблицы:

  • <colgroup> — контейнер для группы столбцов
  • <col> — определяет стиль отдельного столбца или группы столбцов

21. Что означает значение baseline?

Значение baseline для свойства vertical-align выравнивает базовую линию элемента с базовой линией родителя.

Это особенно полезно для выравнивания текста и inline-элементов.
