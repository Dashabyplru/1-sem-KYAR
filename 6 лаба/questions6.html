Основы HTML и CSS

Ответы на вопросы по веб-разработке

1 Что представляет из себя абсолютная адресация?

Абсолютная адресация - это полный путь к ресурсу в интернете, включая протокол, доменное имя и полный путь к файлу.

http://www.megasite.ru/hzchd/images/labuda.gif

Абсолютные ссылки используются для ссылок на внешние ресурсы и всегда начинаются с указания протокола (http:// или https://).

2 В чем суть относительной адресации?

Относительная адресация указывает путь к ресурсу относительно текущего местоположения документа.

images/labuda.gif

Если текущая страница находится в папке hzchd, то браузер будет искать изображение по пути hzchd/images/labuda.gif.

Относительные пути короче и удобнее при разработке, так как не требуют изменения при переносе сайта на другой домен.
3 Что такое относительный путь от документа? от корневого каталога?

Относительный путь от документа - путь, который отсчитывается от местоположения текущего HTML-документа.

../images/photo.jpg images/icon.png

Относительный путь от корневого каталога - путь, который отсчитывается от корневой папки сайта (начинается с /).

/images/logo.png /css/styles.css
4 Каким образом создать линейный градиент?

Линейный градиент создается с помощью CSS-функции linear-gradient().

background: linear-gradient(направление, цвет1, цвет2, ...);

Примеры:

/* Сверху вниз */ background: linear-gradient(blue, red); /* Слева направо */ background: linear-gradient(to right, blue, red); /* Диагональный градиент */ background: linear-gradient(45deg, blue, red); /* С несколькими цветами */ background: linear-gradient(to right, red, orange, yellow, green, blue);
5 Поясните, как создать радиальный градиент.

Радиальный градиент создается с помощью CSS-функции radial-gradient().

background: radial-gradient(форма размер позиция, цвет1, цвет2, ...);

Примеры:

/* Простой радиальный градиент */ background: radial-gradient(red, blue); /* Круглый градиент в центре */ background: radial-gradient(circle, red, blue); /* Эллиптический градиент */ background: radial-gradient(ellipse, red, blue); /* С указанием позиции */ background: radial-gradient(circle at top left, red, blue);
6 Каким образом создать обтекание изображения текстом?

Обтекание изображения текстом создается с помощью CSS-свойства float.

img { float: left; /* Обтекание слева */ margin: 10px; /* Отступ вокруг изображения */ }

Также можно использовать:

img { float: right; /* Обтекание справа */ margin: 10px; }
Для отмены обтекания используйте свойство clear: both; на элементе, который должен находиться под обтекаемым элементом.
7 Создайте веб-страницу, на которой обтекание изображения происходит текстом слева.

Пример HTML и CSS для обтекания изображения текстом слева:

<!DOCTYPE html> <html lang="ru"> <head> <meta charset="UTF-8"> <title>Обтекание изображения</title> <style> .container { max-width: 800px; margin: 0 auto; padding: 20px; } .float-left { float: left; margin: 0 15px 15px 0; max-width: 300px; } .clearfix::after { content: ""; display: table; clear: both; } </style> </head> <body> <div class="container clearfix"> <img src="image.jpg" alt="Пример изображения" class="float-left"> <p>Текст будет обтекать изображение слева...</p> <!-- Добавьте больше текста для демонстрации --> </div> </body> </html>
8 Для чего необходимо свойство background-repeat?

Свойство background-repeat определяет, как фоновое изображение будет повторяться на элементе.

Возможные значения:

background-repeat: repeat; /* Повторяется по обеим осям (по умолчанию) */ background-repeat: repeat-x; /* Повторяется только по горизонтали */ background-repeat: repeat-y; /* Повторяется только по вертикали */ background-repeat: no-repeat; /* Не повторяется */ background-repeat: space; /* Повторяется с равными промежутками */ background-repeat: round; /* Повторяется и масштабируется */
Это свойство особенно полезно при использовании небольших фоновых изображений, которые должны заполнять всю область элемента.
9 Какие форматы изображений можно загружать на веб-страницу?

На веб-страницы можно загружать различные форматы изображений:

  • JPEG/JPG - для фотографий и изображений с множеством цветов
  • PNG - для изображений с прозрачностью и графики
  • GIF - для простой анимации и изображений с ограниченной палитрой
  • SVG - векторная графика, масштабируется без потери качества
  • WebP - современный формат с лучшим сжатием
  • BMP - без сжатия, не рекомендуется для веба
Для видеофайлов поддерживаются форматы MP4, WEBM и OGG, как указано в лекции в теге <video>.
10 Для чего используется свойство background-image? Какие значения оно может принимать?

Свойство background-image устанавливает фоновое изображение для элемента.

Значения:

background-image: url("image.jpg"); /* Путь к изображению */ background-image: linear-gradient(red, blue); /* Градиент */ background-image: none; /* Без фонового изображения */ background-image: url("image1.jpg"), url("image2.jpg"); /* Несколько изображений */

Примеры использования:

body { background-image: url("background.jpg"); } .hero { background-image: linear-gradient(rgba(0,0,0,0.5), rgba(0,0,0,0.5)), url("hero-image.jpg"); }
11 Каким образом изменить размер изображения в CSS?

Размер изображения в CSS можно изменить несколькими способами:

/* Для тега img */ img { width: 300px; /* Фиксированная ширина */ height: 200px; /* Фиксированная высота */ max-width: 100%; /* Максимальная ширина */ } /* Для фонового изображения */ .element { background-image: url("image.jpg"); background-size: cover; /* Заполняет весь элемент */ background-size: contain; /* Вписывает в элемент */ background-size: 50% 50%; /* Процентные значения */ background-size: 300px 200px; /* Фиксированные размеры */ }
Используйте object-fit: cover; для тега <img> чтобы изображение заполнило контейнер без искажений.
12 Для чего предназначены префиксы?

Вендорные префиксы используются для добавления экспериментальных или нестандартных CSS-свойств в разных браузерах.

Основные префиксы:

-webkit- /* Chrome, Safari, новые версии Opera */ -moz- /* Firefox */ -ms- /* Internet Explorer */ -o- /* Старые версии Opera */

Пример использования:

.element { -webkit-border-radius: 10px; -moz-border-radius: 10px; border-radius: 10px; -webkit-transition: all 0.3s; -moz-transition: all 0.3s; transition: all 0.3s; }
Современные браузеры все меньше требуют префиксы, так как многие свойства уже стандартизированы.
13 Каким образом зафиксировать фоновое изображение?

Чтобы зафиксировать фоновое изображение (оно не будет прокручиваться с контентом), используйте свойство background-attachment.

body { background-image: url("background.jpg"); background-attachment: fixed; /* Фиксированное положение */ }

Другие значения:

background-attachment: scroll; /* Прокручивается с контентом (по умолчанию) */ background-attachment: local; /* Прокручивается с содержимым элемента */ background-attachment: fixed; /* Фиксировано относительно окна просмотра */
14 Для чего используется свойство background-blend-mode? Назовите значения свойства.

Свойство background-blend-mode определяет, как фоновое изображение должно смешиваться с фоновым цветом или другими фоновыми изображениями.

Основные значения:

background-blend-mode: normal; /* Обычное наложение */ background-blend-mode: multiply; /* Умножение */ background-blend-mode: screen; /* Экран */ background-blend-mode: overlay; /* Перекрытие */ background-blend-mode: darken; /* Затемнение */ background-blend-mode: lighten; /* Осветление */ background-blend-mode: color-dodge; /* Осветление основы */ background-blend-mode: color-burn; /* Затемнение основы */ background-blend-mode: difference; /* Разница */ background-blend-mode: exclusion; /* Исключение */

Пример:

.element { background-image: url("image.jpg"); background-color: blue; background-blend-mode: multiply; }
15 С какой целью применяется transparent?

Значение transparent используется для указания полностью прозрачного цвета.

Примеры применения:

/* Прозрачный фон */ .element { background-color: transparent; } /* Прозрачная граница */ .button { border: 2px solid transparent; } /* Прозрачный цвет текста */ .text { color: transparent; }
transparent является синонимом для rgba(0,0,0,0) - полностью прозрачного черного цвета.
16 Что такое карта-изображение?

Карта-изображение (image map) - это изображение с определенными активными областями, при нажатии на которые происходит активация ссылок.

Создается с помощью тегов:

<img src="image.jpg" alt="Карта" usemap="#mapname"> <map name="mapname"> <area shape="rect" coords="x1,y1,x2,y2" href="link1.html" alt="Описание"> <area shape="circle" coords="x,y,radius" href="link2.html" alt="Описание"> <area shape="poly" coords="x1,y1,x2,y2,x3,y3,..." href="link3.html" alt="Описание"> </map>

Как указано в лекции, карта изображений создает зоны обычного изображения, при нажатии на которые происходит активация ссылок.

17 Как сделать вид маркера графическим?

Чтобы сделать вид маркера списка графическим, используйте свойство list-style-image.

ul { list-style-image: url('marker.png'); }

Или с помощью псевдоэлементов для большего контроля:

ul { list-style: none; } li::before { content: ""; display: inline-block; width: 16px; height: 16px; background-image: url('marker.png'); background-size: contain; margin-right: 10px; }
18 Для чего необходимо свойство background-origin?

Свойство background-origin определяет область позиционирования фонового изображения.

Значения:

background-origin: padding-box; /* Отступы включены (по умолчанию) */ background-origin: border-box; /* Граница включена */ background-origin: content-box; /* Только содержимое */

Пример:

.element { border: 10px solid rgba(0,0,0,0.3); padding: 20px; background-image: url('pattern.png'); background-origin: content-box; background-repeat: no-repeat; }
Это свойство работает в сочетании с background-position и определяет, от какой части элемента отсчитывается позиция фона.
