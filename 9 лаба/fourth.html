grid-template-rows
grid-gap
grid-area

grid-template-columns определяет количество и размеры колонок в grid-контейнере.

Можно использовать различные единицы измерения: пиксели (px), проценты (%), дробные единицы (fr), auto.

Пример: grid-template-columns: 1fr 2fr 1fr;

grid-template-rows определяет количество и размеры строк в grid-контейнере.

Аналогично columns, но применяется для строк. Можно комбинировать с columns для создания сложных сеток.

Пример: grid-template-rows: 100px auto 200px;

grid-gap (или gap в современном синтаксисе) задает расстояние между элементами grid-сетки.

Можно задать одно значение для всех промежутков или два значения отдельно для row-gap и column-gap.

Пример: grid-gap: 20px 15px;

grid-area позволяет задать имя области или определить расположение элемента в сетке.

Можно использовать для размещения элемента в определенной области или для задания границ элемента.

Пример: grid-area: 2 / 1 / 4 / 3;
