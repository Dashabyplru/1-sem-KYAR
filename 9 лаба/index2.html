CSS (англ. Cascading Style Sheets, что переводится как «каскадные таблицы стилей») — это формальный язык, предназначенный для описания внешнего вида и визуального оформления документа, в первую очередь веб-страницы, которая обычно создается с использованием языка разметки, такого как HTML или XHTML. Основная цель и философия CSS заключается в четком разделении и отделении описания логической структуры веб-страницы от описания её внешнего вида, визуального представления и стилевого оформления. Такой подход значительно увеличивает доступность документа для различных пользователей и устройств, предоставляет разработчикам гораздо большую гибкость в управлении представлением контента, а также эффективно уменьшает сложность и повторяемость в структурном содержимом веб-страниц, делая код более чистым, понятным и легким в поддержке.
CSS используется веб-разработчиками для задания цветов, шрифтов, стилей, расположения блоков и других аспектов представления веб-страниц. Кроме того, CSS позволяет представлять один и тот же документ в различных стилях или методах вывода: для экранного отображения, для печати, для чтения голосом или при выводе устройствами, использующими шрифт Брайля. CSS применяется также к любым XML-документам, например, к SVG.
CSS — одна из технологий, одобренных консорциумом W3C и получивших название «стандарты Web». В 1990-х стала ясна необходимость стандартизировать Web. Термин «каскадные таблицы стилей» предложил Хокон Ли в 1994 году. Совместно с Бертом Босом он развивал CSS. В отличие от многих языков стиля того времени, CSS использует наследование от родителя к потомку. W3C проявил интерес к CSS, и в декабре 1996 года была издана рекомендация CSS1.
